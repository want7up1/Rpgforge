第 7 回合 · 市第三人民医院三楼休息室 · 傍晚

Action · 自由行动

你让苏婉清留在休息室反锁房门，自己前往地下室重启医院供电。

GM · DeepSeek Pro · streamed

林辰将破晓的微光压低，转身握住 苏婉清的手腕，将一支肾上腺素注射器轻轻拍进她掌心。

“反锁房门，只给我留暗号。”他声音压得极低，“如果我遇到麻烦， 就用这根注射器敲水管，三长两短。”

备用楼梯藏在医院西翼最深处。消防门虚掩，推开时刮出一阵铁锈味的冷风。 越往下，空气越湿，踩到第三段台阶时，积水突然没过鞋底。

地下室铁门半开，柴油发电机沉默如巨兽。就在灯管重新亮起时， 冷藏室门后传来极其微弱的撞击，像冻僵的手指在叩门。

他沿原路返回三楼，门锁咔嗒一声打开。苏婉清听见远处发电机的低鸣， 眼中迸出希望：“你做到了？”

建议行动

A 从药房正门进入，趁尸群尚未突破铁门快速清理堵门丧尸。
B 从污物电梯井潜入药房，避开正面威胁但承担污染风险。
C 返回地下室调查锁住的冷藏室，寻找晨星公司留下的线索。
D 让苏婉清警戒，自己独自取药，以最快速度完成清单。
